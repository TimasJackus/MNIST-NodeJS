# MNIST-NodeJS
Digit recognition with neural network

## Installation

Download and install [yarn](https://yarnpkg.com/en/docs/getting-started), [NodeJS](https://nodejs.org/en/download/):

Install npm packages:

```
yarn
```

## Development

Run in one terminal:

```
gulp
```

Run in second terminal:

```
yarn watch
```

## Production

Run:

```
yarn start
```
